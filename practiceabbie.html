Style & Explain the CSS Box Model

The box model is something you will use over and over. It's critical to understand and utilize spacing for effective design. Use this space to describe the box model, insert an image that supports your written description, and include two examples where you have demonstrated the usefulness of spacing and what the box model can do. Use subheaders like "h3" to organize your work and descriptions.

ANSWER

The CSS Box Model says that all HTML elements can be considered boxes and we can apply style and design to those boxes. Each box has a margin, a border, some padding, and then the content of the HTML element. You can use the box model to make sure elements are properly sized and spaced to create the style and design aesthetic you are going for.

How to Insert CSS

Briefly define external, embedded, and inline CSS and where you would use each. Next, utlize embedded CSS to style the background and font of a "div" element. The content of "div" should be your description of where to insert CSS. Identify how the syntax changes when using inline versus embedded.

ANSWER

CSS can be inserted into our webpages in a few different ways:
  • Inline Styles: Use within the HTML tag to apply unique style to a single element. Use sparingly as you lose the efficiency advantages to other CSS usage types. If you use this method, Internal and External CSS applications will be ignored.
  • Internal CSS: When CSS coding is embedded in the "head" section of the HTML document, this is called Internal CSS. Use this when you have specific style that only applies to the one page. Again, try to use sparingly as you will have to maintain this style within this .html doc specifically.
  • External Style Sheet: Here is where most of your styling will occur. This is the most efficient way to manage style that applies to an entire website and all its subpages. Make one change in the external styles.css sheet and it will apply to your whole site!
  • Browser Defaults: Last but not least, the browser will fall back to its defaults if you do not declare other styles using one of the prior three methods.
